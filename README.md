# ResNet-decoder in PyTorch
ResNet decoder using transposed ResNet (resnet 50, resnet 101)

## Prerequisites

1. Install [Pytorch](https://pytorch.org/)
1. Install torchsummary: ```pip install torchsummary```

## Usage
Run 
```Python test.py```
